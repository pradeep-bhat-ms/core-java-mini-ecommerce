# Core Java Mini E-Commerce System

## Project Description
This is a simple console-based Mini E-Commerce System developed using Core Java.  
The project simulates basic online shopping functionalities.

---

## Features
- View Products
- Add Product to Cart
- Remove Product from Cart
- View Cart Items
- Calculate Total Bill
- Place Order

---

## Technologies Used
- Core Java
- Eclipse IDE

---

## Concepts Used

- Constructor
- Encapsulation
- Methods
- ArrayList
- Conditional Statements
- Loops
- Scanner Class
